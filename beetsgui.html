# Udfyld felterne ovenfor
Indsæt i terminal: beet config -e — åbner config-filen i din editor.
Discogs ▲
1. Log ind på Discogs → Settings → Developer
2. Klik "Generate new token" → kopier token
OAuth sker automatisk når du kører beet import første gang med discogs-pluginet aktivt. Beets åbner en browser og gemmer token i ~/.config/beets/.
⚠ Discogs-token giver fuld adgang til din konto — hold den privat.
MusicBrainz ▼
⚠ Kodeord gemmes i plain text i config.yaml. Opret konto: musicbrainz.org
Beatport4 ▼
Kræver: pipx inject beets beets-beatport4
Pluginet skal hedde beatport4 i config (ikke beatport).
1. Log ind på beatport.com i Chrome/Edge
2. Åbn Developer Tools (Cmd+Opt+I) → Network
3. Filtrer på "token" → find POST-request → kopiér Response-JSON
4. Gem som ~/.config/beets/beatport_token.json
Token læses automatisk af pluginet.
⚠ Login-oplysninger gemmes i plain text i config.yaml.
Vejledning: beet version
beet version (ikke --plugins) viser aktive plugins.
Plugins aktiveres kun når de er listet under plugins: i config.yaml.
⚠ Kør aldrig beet config -e mens GUI redigerer.
